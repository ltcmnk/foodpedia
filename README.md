# Foodpedia 🍽️📖  
**Foodpedia** é uma aplicação web voltada para a exploração e descoberta de receitas e alimentos, oferecendo uma interface simples e intuitiva para consultar pratos, ingredientes e informações culinárias.  

O projeto foi desenvolvido com foco em organização de conteúdo e boa experiência de navegação, funcionando como uma espécie de **enciclopédia gastronômica digital**.

---

## 📌 Funcionalidades

- Navegação por receitas e conteúdos culinários
- Visualização de detalhes de pratos e ingredientes
- Interface amigável e responsiva
- Organização do conteúdo em formato de catálogo/enciclopédia
- Estrutura simples e fácil de executar localmente

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **JavaScript**

## 🚀 Como Rodar o Projeto

### 1️⃣ Clone o repositório

```bash
git clone https://github.com/ltcmnk/foodpedia.git
cd foodpedia
````

### 2️⃣ Execute o projeto

Abra o arquivo `index.html` diretamente no navegador:

* Clique duas vezes no `index.html`

ou utilize o **Live Server** no VSCode:

1. Abra a pasta do projeto no VSCode
2. Clique com o botão direito em `index.html`
3. Selecione **Open with Live Server**

## 🎯 Objetivo do Projeto

O objetivo do Foodpedia é fornecer uma plataforma simples e organizada para usuários explorarem receitas e informações gastronômicas, promovendo aprendizado culinário e descoberta de novos pratos.
